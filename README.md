# KTS Official Website 2026

This is a [Next.js](https://nextjs.org) project for the KTS Official Website.

## Getting Started Locally

To run this project on your local system, follow these steps:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Install Dependencies
Navigate to the project directory and install the necessary dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

### 3. Run the Development Server
Start the local development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### 4. View the App
Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

You can start editing the page by modifying `app/page.tsx`. The page automatically updates as you edit the file.
